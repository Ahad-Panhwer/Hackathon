## 🏆 How It Works
1. Pick a theme from the [Themes file](THEMES.md).  
2. Form a team (2–3 recommended).  
3. Build your project (fork → create → submit).  
4. Submit using the **Submission Form (GitHub Issue)**.  
5. Judges will review your work.
